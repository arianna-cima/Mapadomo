Sources and definitions

Headline price data sources for DE

2013 to 2017 prices are based on purchased data, available per postcode and building type. This prices are weighted by municipal stocks data by year. Prices are extended forward using the Immoscout24.de ndex by NUTs-region. Prices are extended backward to 2004 based on vdp indexes for the 7 major cities. For all other regions, prices are extended backward to 2010 by using RWI index calcuations (for 141 regions). Finally, since house prices did not change by more than 10% 2006-2010, all prices are extended backward from 2010 to 2006 with national vdp indexes for flats and houses.


Offer price data sources for DE

Webscraped immoscout24.de in 2018, extended forward and backward with the price index from immoscout24.de


Building stock data sources for DE

dwellings and sqm per type all stem from destatis.de (dataset Gebaude- und Wohnungszahlung), adjusted for a structural break in 2011, using reported NUTS-3 aggreagtes data for all years since 1999.


Definition of house v apartment for DE

house is any dwelling within a building that contains 1 or 2 dwellings.

Definition and data sources for square metres for DE

Square metre is useful square metre as defined by destatis.de
