R_codes
SE_dwelling_timeseries
It creates the number of dwelling at municipality level from 1990 to 2019

SE_size
It creates the variable size according to the number of dwelling for each size cluster

SE_rooms 
It creates the number of rooms at municipality level

SE_total_m2
It creates the number of square meters at municipality level in 2000, 2005, 2010, 2015

SE_interpolation
interpolate m2 missing years according to the m2/dwelling rate and extending the time series multiplying for the number of dwellings

SE_hpi
It downloads oecd house price index data 

SE_price
It downloads webscraped price and it creates mean and median at each nuts level
